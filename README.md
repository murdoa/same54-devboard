# SAM-E54 Development Board
## Features
- USB Host / Device
- USB Serial (CP2102) 
- SDHC Interface
- 100BASE-T Ethernet
- QSPI with W25N01GV footprint (1Gb flash)
- Cortex M ISP header
- 4 Indicator LEDs
- 3 Push Button switches
- 43 Pins accessible through 60 pin header
